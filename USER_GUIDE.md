## Getting Started

### Installation
1. Download the extension
2. Open Chrome and go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the BrowseRhythm folder

### First Use
Once installed, BrowseRhythm automatically starts tracking your browsing activity. Click the extension icon to view your stats!

### üéØ Focus Mode
Block distracting websites to stay productive:

1. Click the extension icon
2. Toggle "Focus Mode" ON
3. Click the settings icon (gear)
4. Select categories to block (Social, Streaming, News, Gaming)
5. Or add custom sites to block

**Preset Categories:**
- **Social Media** (50+ sites): Facebook, Twitter, Instagram, TikTok, etc.

## FAQ

**Q: Does BrowseRhythm track incognito/private browsing?**
A: No, browser extensions don't have access to incognito windows by default.

**Q: How long is data stored?**
A: Data is automatically cleaned up after 30 days to save space.

**Q: Can I export my data?**
A: Yes! Use the data export feature in settings (coming soon).

**Q: Does Focus Mode work on all websites?**
A: Focus Mode blocks navigation to blocked sites. Some sites may load if already open.

**Q: How accurate is the time tracking?**
A: Very accurate! Time is tracked when a tab is active and visible.

**Q: Can I use this on Firefox?**
A: Currently Chrome only. Firefox support may come in the future.

## Troubleshooting

### Extension not tracking time
1. Reload the extension in `chrome://extensions`
2. Check that you've granted necessary permissions
3. Look for errors in the console

### Sites not blocking
1. Ensure Focus Mode is enabled
2. Check that the category is selected
3. Verify the domain is in your blocklist

### Data not showing
1. Browse some websites first
2. Wait at least 30 seconds for data to save
3. Reload the popup
